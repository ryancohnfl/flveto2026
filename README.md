# Florida FY2026-27 Budget Veto Explorer

An interactive, self-contained explorer of Gov. Ron DeSantis's line-item vetoes of HB 5001-E,
signed June 29, 2026. Filter, sort, and search all 621 vetoed line items by recipient type,
county, sector, dollar amount, and business-development priority.

- **Live site:** https://ryancohnfl.github.io/flveto2026/ (after enabling GitHub Pages)
- **Source data:** the Governor's official veto transmittal (39 pp.), parsed and cross-checked
  against the enacted budget and the Governor's stated totals.
- Single self-contained file (`index.html`); brand fonts and data are embedded, no dependencies.

Prepared by Sachs Media.
